Medidas relativas

Usar las herramientas de desarrollador para ver esta página en diferentes tamaños.

width: 50%

Siempre tomará de referencia su padre (el body en este caso).

width: 50vw

Siempre tomará de referencia el viewport (que en este caso particular tiene el mismo ancho del body).

width: 800px

Los contenedores siempre
tendrán el mismo ancho.

width: 50%

Siempre tomará de referencia su padre (el contenedor turquesa).

width: 800px

Los contenedores siempre
tendrán el mismo ancho.

width: 50vw

Siempre tomará de referencia el viewport.
